# BluePad
 
## はじめに
 
BluePadへようこそ。
あなたの新しいマークダウンエディターです。
高速でクリーン、シンプルさを追求。
 
```javascript
const editor = new BluePad();
editor.write("こんにちは！");
```
機能

必要なものだけを、
不要なものなしで。

スピードと明確さのために設計。Electronなし — Tauriベースの高速ネイティブアプリ。

✎

WYSIWYG + ソースモード

リッチテキスト編集とマークダウンソースをショートカット一つで切り替え。フォーマットをリアルタイムで確認または構文を直接編集。

📄

マルチタブ編集

クリーンなタブインターフェースで複数のドキュメントを同時に作業。位置を失わずに素早くコンテキストを切り替え。

🎨

4つの美しいテーマ

クラシックライト、ソフトセピア、ダークモード、ディープナイト。気分と環境に合ったパレットを選択。

🕶

フォーカスモード

現在の段落以外をすべて淡く表示。気が散る要素を排除し、執筆のフローに集中。

🔍

検索 & 置換

正規表現対応の強力な検索。ドキュメント内のテキストを見つけて即座に置換。

🌐

多言語対応

英語、韓国語、日本語のUI完全対応。CJKテキストレンダリングでどの言語でも執筆可能。
