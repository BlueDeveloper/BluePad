インストール手順

🖥️ Windows

  1. ダウンロードしたBluePad-latest_x64.msiを実行
  2. SmartScreen警告時は詳細情報 → 実行
  3. インストーラーの指示に従う
  4. スタートメニューからBluePadを起動
コード署名は未実装。SmartScreen警告が表示されますが運用方針として受容しています — 安全です。

🐧 Linux

  1. AppImage: chmod +x BluePad-latest_amd64.AppImage後に実行
  2. .deb: sudo apt install ./BluePad-latest_amd64.deb
  3. .rpm: sudo dnf install ./BluePad-latest-1.x86_64.rpm
  4. またはsudo rpm -iを使用
Linux版は自動アップデート未対応。新バージョン公開時はこのページから再取得してください。
