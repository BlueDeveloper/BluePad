Getting Started

How do I install BluePad?
Download the MSI file from bluepad.work and run it. If Windows SmartScreen shows a warning, click "More info" then "Run anyway". BluePad is safe — it's built on open-source technology.
What's the difference between Free and Pro?
Free: Up to 3 tabs, Classic + Dark theme, JSON/YAML editing, auto-save, find & replace.
Pro ($10.99, one-time): Unlimited tabs, all 5 themes, focus mode, HTML/PDF export. 14-day free trial included.
How do I activate a Pro license?
Go to Settings > Activate License, paste your license key (format: BP-XXXX-XXXX-XXXX-XXXX), and click Activate. You can use it on up to 3 devices.

Features

Which file formats are supported?
Markdown: .md, .markdown, .mdx (WYSIWYG editor)
JSON: .json, .jsonc (syntax highlighting + auto-format)
YAML: .yaml, .yml (syntax highlighting + auto-format)
Text: .txt, .log, .env, .ini, .conf
How do I auto-format JSON/YAML?
Open a .json or .yaml file, then press Ctrl+Shift+F or go to Edit > Format.
How do I use the dark theme?
Go to View > Theme: Dark. Dark and Classic themes are free for all users.
Does BluePad save my work automatically?
Yes. Auto-save runs every 30 seconds for files that have been saved before. You can toggle it in File > Auto Save. All settings, open tabs, and sidebar state persist when you restart the app.
How do code files like .js / .html / .css work?
BluePad opens code files in a CodeMirror-based editor with syntax highlighting and line numbers. Supported extensions include .js / .jsx / .mjs / .cjs / .ts / .tsx, .html / .htm / .xhtml, and .css / .scss / .sass / .less. Colors adapt to your selected theme automatically.
What is the outline sidebar and how do I reorder chapters?
Press Ctrl+Shift+L to open the outline sidebar. It shows all H1~H6 headings in your document. Drag and drop any heading onto another to move the entire section — heading plus all its content and child headings — to a new position. Useful for novels and long-form essays where you need to restructure chapters.

Writing Mode

What is Writing Mode and how do I turn it on?
Writing Mode (Pro) is a focused environment built for novelists, bloggers, and journal writers. Press Ctrl+Shift+W in any markdown file. A panel appears on the right with daily word goals, a session timer, real-time WPM, a 30-day heatmap, a sprint timer (10/15/25/45 min), and sentence analysis (long sentences, passive voice, adverbs, filler words, readability).
What do Typewriter Mode and Highlight Mode do?
When Writing Mode is on, the editor automatically applies two visual changes. Typewriter Mode keeps the current line fixed in the middle of the screen as you type — your eyes never move down to the bottom of the page. Highlight Mode dims every paragraph except the one your cursor is in, so you only see what you are writing now.
How are sentence analysis numbers calculated?
All analysis runs locally — nothing leaves your computer. The threshold for a "long" sentence is 25 words. Passive voice detection uses simple patterns (English be-verb + past participle, Korean "되다/받다/지다" endings). Adverb detection looks for English -ly words and Korean "~게/~히/매우/너무" patterns. Numbers that exceed reasonable thresholds turn orange — treat them as gentle reminders, not strict rules.

Troubleshooting

Windows SmartScreen blocks the installer
This happens because BluePad doesn't have a code signing certificate yet. Click "More info" then "Run anyway". The app is safe.
How do I request a refund?
Go to bluepad.work/support and select "Refund Request". Include your email and we'll process it within 1-2 business days.
How do I move my license to another computer?
On the old computer: Settings > Activate License > Deactivate. Then activate on the new computer with the same key. Max 3 devices.
What does the 14-day trial include?
All Pro features are unlocked during the trial: unlimited tabs, all 5 themes, focus mode, HTML/PDF export, and Writing Mode with the full statistics panel. No signup, no credit card. After 14 days, the editor reverts to Free unless you activate a license.
Does BluePad notice when files change outside the editor?
Yes. When you switch back to BluePad (window focus), it compares the disk mtime to what it last read. If the file was modified externally, you are offered to reload. Useful when you edit the same file in another tool (e.g., git pull, VS Code).
What if my file is not UTF-8 encoded?
BluePad detects non-UTF-8 files (CP949, EUC-KR, Shift-JIS) by looking for replacement characters and falls back to the appropriate decoder. UTF-8 BOM is stripped on save so files remain clean.
