# BluePad
 
## Getting Started
 
Welcome to BluePad, your new favorite
markdown editor. It's fast, clean, and
built for writers who value simplicity.
 
```javascript
const editor = new BluePad();
editor.write("Hello, world!");
```
Features

Everything you need,
nothing you don't.

Built for speed and clarity. No bloat, no Electron — just a fast native app powered by Tauri.

✎

WYSIWYG + Source Mode

Switch between rich text editing and raw markdown source with a single shortcut. See your formatting live or work directly with syntax.

📄

Multi-tab Editing

Work on multiple documents side by side with a clean tab interface. Quickly switch context without losing your place.

🎨

4 Beautiful Themes

Classic light, soft sepia, dark mode, and deep night. Pick the palette that matches your mood and environment.

🕶

Focus Mode

Dim everything except the current paragraph. Eliminate distractions and stay in the flow of your writing.

🔍

Find & Replace

Powerful search with regex support. Find any text across your document and replace it instantly.

🌐

Multi-language

Full UI support for English, Korean, and Japanese. Write in any language with proper CJK text rendering.
