Welcome to BluePad.

```javascript
const editor = new BluePad();

```
Features

Only what you need.
Nothing you don't.

A native app built on Tauri, not Electron. Fast from launch, lighter the more you use it.

✎

WYSIWYG + Source

Switch between rich editing and Markdown source with a single shortcut.

◥

Multi-Tab

Manage multiple documents in tabs. Drag to reorder, Ctrl+Tab to switch.

◎

4 Themes

Classic, Blue, Red, Polarity. Choose the palette that fits your mood.

⌇

Focus Mode

Hide all UI and concentrate on writing. Press ESC to return instantly.

⌕

Find & Replace

Quickly search and instantly replace text within your document.

☯

KO / EN / JA

Three-language UI. Renders CJK text flawlessly.
