Installation

🖥️ Windows

  1. Run the downloaded BluePad-latest_x64.msi
  2. If SmartScreen warns, click More info → Run anyway
  3. Follow the installer wizard
  4. Launch BluePad from the Start menu
Not code-signed yet — SmartScreen will warn. This is intentional during early operations; the binary is safe.

🐧 Linux

  1. AppImage: chmod +x BluePad-latest_amd64.AppImage, then run
  2. .deb: sudo apt install ./BluePad-latest_amd64.deb
  3. .rpm: sudo dnf install ./BluePad-latest-1.x86_64.rpm
  4. Or use sudo rpm -i
Auto-updates are not yet available on Linux. Check this page when a new version ships.
