v1.8.0 Stable 2026-05-08

New Features

  • Dark theme (free for all users)
  • JSON / YAML editing with syntax highlighting and auto-formatting (Ctrl+Shift+F)
  • Always on Top — pin window above all others
  • Selection character count in status bar
  • Reading time estimate in status bar
  • YAML Front Matter display
  • [toc] directive — auto-generated table of contents
  • Highlight CSS support for <mark> tags
  • Feedback page — send suggestions to the developer

Improved

  • Tabs restore on app restart (previously lost)
  • File tree folder persists across restarts
  • Source mode, sidebar, auto-save settings persist
  • Tab close dialog: Save & Close / Close (previously confusing)
  • Checkbox / task list rendering improved
  • Syntax highlighting for code blocks (17 token types)
  • Strikethrough, image, table, blockquote styling enhanced

Security

  • Payment replay protection (CRITICAL fix)
  • Payment amount verification after capture
  • Transaction logging for refund tracking
  • License generation failure recovery
