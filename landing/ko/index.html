# BluePad
 
## 시작하기
 
BluePad에 오신 것을 환영합니다.
여러분의 새로운 마크다운 에디터입니다.
빠르고, 깔끔하며,
심플함을 추구하는 작가를 위해 만들어졌습니다.
 
```javascript
const editor = new BluePad();
editor.write("안녕하세요!");
```
기능

필요한 것만,
불필요한 것은 없이.

속도와 명확함을 위해 설계되었습니다. Electron 없이 — Tauri 기반의 빠른 네이티브 앱.

✎

위지윅 + 소스 모드

리치 텍스트 편집과 마크다운 소스 간 단축키 하나로 전환. 서식을 실시간으로 보거나 구문을 직접 편집할 수 있습니다.

📄

멀티탭 편집

깔끔한 탭 인터페이스로 여러 문서를 동시에 작업. 작업 위치를 잃지 않고 빠르게 컨텍스트 전환이 가능합니다.

🎨

4가지 아름다운 테마

클래식 라이트, 소프트 세피아, 다크 모드, 딥 나이트. 분위기와 환경에 맞는 팔레트를 선택하세요.

🕶

집중 모드

현재 단락을 제외한 모든 것을 흐리게 처리. 방해 요소를 제거하고 글쓰기 흐름에 집중하세요.

🔍

찾기 & 바꾸기

정규식을 지원하는 강력한 검색. 문서 내 텍스트를 찾고 즉시 교체할 수 있습니다.

🌐

다국어 지원

영어, 한국어, 일본어 완벽 지원 UI. CJK 텍스트 렌더링으로 어떤 언어로든 작성 가능.
