# BluePad
 
## 시작하기
 
BluePad에 오신 것을 환영합니다.
글쓰기에만 집중할 수 있도록 설계되었습니다.
 
```javascript
const editor = new BluePad();
editor.write("안녕하세요");
```
기능

필요한 것만.
불필요한 것은 없이.

Electron 없이, Tauri 기반 네이티브 앱. 시작부터 빠르고, 사용할수록 가볍습니다.

✎

위지윅 + 소스

리치 편집과 마크다운 소스를 단축키 하나로 전환합니다.

◥

멀티탭

여러 문서를 탭으로 관리. 앱 재시작 시 이전 탭 자동 복원.

◎

5가지 테마

Classic, Dark, Blue, Red, Polarity. 다크 모드 무료 지원.

⌇

집중 모드

모든 UI를 숨기고 글쓰기에만 집중. ESC로 즉시 복귀.

{}

JSON / YAML 편집

구문 강조 + Ctrl+Shift+F 자동 정렬. 개발자를 위한 편의 기능.

⌕

찾기 & 바꾸기

문서 내 텍스트를 빠르게 검색하고 즉시 교체합니다.

☯

한/영/일

3개 언어 UI. CJK 텍스트를 완벽하게 렌더링합니다.

📌

항상 위에 + 상태 유지

창 고정, 폴더/사이드바/테마 설정이 앱 재시작 후에도 유지됩니다.

FAQ

자주 묻는 질문

BluePad는 무료인가요?

네. 기본 편집, 멀티탭, 위지윅·소스 모드, JSON·YAML 편집, 자동 저장 등 일상 글쓰기 기능은 무료로 제공합니다. 무제한 탭, 전체 테마, 집중 모드, HTML·PDF 내보내기 같은 고급 기능은 Pro($10.99, 일회성 결제·평생 소유)에서 제공하며 14일 무료 체험이 가능합니다.

BluePad와 Typora의 차이는 무엇인가요?

BluePad는 Electron이 아닌 Tauri(Rust) 기반이라 설치 용량이 약 4~7MB로 가볍고 실행이 빠릅니다. 위지윅 편집을 무료로 제공하며, Pro도 구독이 아닌 일회성 결제로 평생 사용합니다.

Windows 외에 Linux도 지원하나요?

네. Windows 10·11용 MSI와 Linux용 AppImage·deb·rpm을 제공합니다. (현재 macOS는 미지원입니다.)

설치 시 SmartScreen 경고가 나오는데 안전한가요?

코드 서명 인증서가 아직 적용되지 않아 Windows에서 경고가 표시될 수 있습니다. "추가 정보 → 실행"으로 설치할 수 있으며, BluePad는 오픈소스 기반으로 악성코드를 포함하지 않습니다.

Pro 라이선스는 구독인가요?

아니요. Pro는 $10.99 일회성 결제이며 평생 업데이트가 포함됩니다. 월·연 구독료가 없습니다.
