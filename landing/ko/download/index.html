BluePad를 설치하세요

Windows와 Linux를 지원합니다. 운영체제를 자동 감지해 권장 패키지를 강조합니다.

감지된 OS: —
권장
Windows
Windows 10 / 11 · 64-bit
코드 서명이 없어 SmartScreen 경고가 표시될 수 있습니다. "추가 정보 → 실행"으로 진행하세요.
MSI 다운로드 · 7 MB
권장
Linux
x86_64 · AppImage / deb / rpm
AppImage는 별도 설치 없이 실행 가능합니다. Debian/Ubuntu는 deb, Fedora/RHEL은 rpm을 권장합니다.
AppImage 다운로드 · 80 MB
.deb (Debian/Ubuntu) .rpm (Fedora/RHEL)
macOS는 아직 지원되지 않습니다. 향후 지원 예정 — 알림을 원하시면 피드백으로 알려주세요.

설치 안내

🖥️ Windows

  1. 다운로드한 BluePad-latest_x64.msi 실행
  2. SmartScreen 경고 시 추가 정보 → 실행
  3. 설치 마법사 따라 진행
  4. 시작 메뉴에서 BluePad 실행
코드 서명이 없어 SmartScreen 경고가 표시됩니다. 운영 방침으로 수용 — 신뢰할 수 있습니다.

🐧 Linux

  1. AppImage: chmod +x BluePad-latest_amd64.AppImage 후 실행
  2. .deb: sudo apt install ./BluePad-latest_amd64.deb
  3. .rpm: sudo dnf install ./BluePad-latest-1.x86_64.rpm
  4. 또는 sudo rpm -i 사용
Linux판은 자동 업데이트가 아직 지원되지 않습니다. 새 버전이 나오면 이 페이지에서 다시 받아주세요.
