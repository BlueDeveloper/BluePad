- **deploy.sh [7/8] 검증 단계 `set -e` 즉사 수정 (만성 이슈 해결)**

  - 수정: HTTP 코드를 `curl -s -o /dev/null -w '%{http_code}'`로 받아 파이프 제거, update.json grep은 `|| true` 가드. 커밋 `dbc210e`.
  - v1.16.0은 이 수정 전 배포라 검증에서 멈췄으나 **사용자 대상 배포물은 전부 정상**(MSI HTTP 200, update.json 1.16.0 서명 포함). 누락된 IndexNow(200)·GitHub Release(v1.16.0, MSI 첨부)는 수동 복구 완료.
