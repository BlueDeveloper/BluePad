# BluePad 기능정의서

> **버전:** 1.0.0 | **최종 업데이트:** 2026-05-07

---

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| 제품명 | BluePad |
| 카테고리 | 마크다운 에디터 (Typora 대안) |
| 기술 스택 | Tauri v2 (Rust) + React + TypeScript + Milkdown + CodeMirror 6 |
| 플랫폼 | Windows (MSI) |
| 수익 모델 | Freemium (Free + Pro 라이선스) |
| 다국어 | 한국어, 영어, 일본어 (146개 번역 키) |

---

## 2. 에디터

### 2.1 편집 모드 (3종)

| 모드 | 대상 파일 | 엔진 | 설명 |
|------|-----------|------|------|
| WYSIWYG | .md, .markdown, .mdx | Milkdown (ProseMirror) | 실시간 렌더링 마크다운 편집 |
| 소스 모드 | .md (Ctrl+/ 토글) | textarea | 원본 마크다운 텍스트 직접 편집 |
| 코드 에디터 | .json, .yaml, .yml | CodeMirror 6 | 구문 강조 + 자동 정렬 |

### 2.2 마크다운 지원 문법

| 문법 | 지원 | 비고 |
|------|------|------|
| 제목 (H1~H6) | O | |
| **굵게** / *기울임* / ~~취소선~~ | O | GFM |
| 인라인 코드 / 코드 블록 | O | Prism.js 구문 강조 |
| 인용구 | O | |
| 순서 목록 / 비순서 목록 | O | |
| 체크리스트 | O | GFM |
| 표 (Table) | O | GFM |
| 수평선 | O | |
| 링크 / 이미지 | O | |
| LaTeX 수식 | O | KaTeX ($inline$, $$block$$) |
| Mermaid 다이어그램 | O | 코드 블록 내 SVG 렌더링 |

### 2.3 이미지 처리

- 클립보드 붙여넣기 (Ctrl+V) → Base64 data URL 삽입
- 드래그 & 드롭 → Base64 data URL 삽입

### 2.4 코드 에디터 기능 (JSON/YAML)

- 구문 강조
- 줄 번호
- 활성 줄 하이라이트
- 괄호 매칭
- 코드 접기 (fold gutter)
- 자동 정렬 (Ctrl+Shift+F)
  - JSON: `JSON.stringify(parsed, null, 2)`
  - YAML: `js-yaml` 라이브러리
- 검색 (CodeMirror 내장)
- 줄 바꿈 (line wrapping)

## 3. 파일 관리

### 3.1 지원 파일 형식

| 확장자 | MIME | 에디터 | 파일 연결 |
|--------|------|--------|-----------|
| .md, .markdown, .mdx | text/markdown | Milkdown | O |
| .json, .jsonc | application/json | CodeMirror | O |
| .yaml, .yml | application/x-yaml | CodeMirror | O |
| .txt, .log | text/plain | textarea | O |
| .env, .ini, .conf, .cfg, .properties | - | textarea | X |

### 3.2 파일 작업

| 기능 | 단축키 | 설명 |
|------|--------|------|
| 새 파일 | Ctrl+N | 빈 마크다운 탭 생성 |
| 열기 | Ctrl+O | 복수 파일 선택 가능 |
| 저장 | Ctrl+S | 경로 없으면 다른이름저장 |
| 다른이름저장 | Ctrl+Shift+S | |
| 자동 저장 | - | 30초 간격, 토글 가능, 기본 ON |
| 최근 파일 | - | 최근 10개 기록, 메뉴에서 열기 |
| CLI 열기 | - | 명령줄 인수로 파일 경로 전달 |

### 3.3 멀티탭

| 기능 | 설명 |
|------|------|
| 탭 생성 | 파일 열기 또는 Ctrl+N |
| 탭 전환 | 클릭 또는 Ctrl+Tab / Ctrl+Shift+Tab |
| 탭 닫기 | X 버튼, 중클릭, Ctrl+W |
| 탭 리오더 | 드래그 & 드롭 |
| 수정 표시 | 탭 이름 앞 점(dot) 표시 |
| 미저장 경고 | 닫기 시 "저장하시겠습니까?" 다이얼로그 |
| 탭 제한 | Free: 최대 3개 / Pro: 무제한 |

## 5. UI 구성

### 5.1 메뉴바

| 메뉴 | 주요 항목 |
|------|-----------|
| 파일 | 새 파일, 열기, 저장, 다른이름저장, 자동 저장, 최근 파일, HTML/PDF 내보내기 |
| 편집 | 찾기, 바꾸기, 자동 정렬 (json/yaml만 표시) |
| 보기 | 소스 모드, 개요, 파일트리, 집중 모드, 글꼴 크기, 테마, 목표 단어수 |
| 설정 | 언어 선택, 라이선스, 고객지원, 업데이트 확인, 정보 |

### 5.2 포맷 툴바 (WYSIWYG 전용)

굵게 | 기울임 | 취소선 | 인라인 코드 | H1 | H2 | H3 | 순서 목록 | 비순서 목록 | 인용 | 코드 블록 | 표 | 수평선 | 링크

### 5.3 토글 버튼 (메뉴바 우측)

| 버튼 | 기능 | 단축키 |
|------|------|--------|
| `</>` | 소스 모드 | Ctrl+/ |
| 폴더 | 파일 트리 | - |
| 삼선 | 개요 사이드바 | Ctrl+Shift+L |
| 집중 | 집중 모드 | F11 |

### 5.4 상태바

파일 경로 | 라이선스 뱃지 (Free/Pro/Trial D-X) | 소스 모드 | 자동 저장 | 목표 진행률 | 글꼴 크기 | 줄 수 | 단어 수 | 글자 수

### 5.5 사이드바

| 사이드바 | 설명 |
|----------|------|
| 개요 (Outline) | H1~H6 계층 트리, 클릭 시 해당 위치 스크롤 |
| 파일 트리 | 폴더 탐색, 파일 클릭으로 열기, 숨김 파일 제외 |

### 5.6 찾기 / 바꾸기

| 기능 | 단축키 |
|------|--------|
| 찾기 열기 | Ctrl+F |
| 바꾸기 열기 | Ctrl+H |
| 다음 찾기 | Enter |
| 이전 찾기 | Shift+Enter |
| 닫기 | Esc |
| 바꾸기 (1건) | 버튼 |
| 모두 바꾸기 | 버튼 |
| 매칭 하이라이트 | 노란색 (현재: 주황색) |

## 8. 라이선스 시스템

### 8.1 플랜 비교

| 기능 | Free | Trial (14일) | Pro |
|------|------|-------------|-----|
| 기본 편집 | O | O | O |
| 자동 저장 | O | O | O |
| Mermaid/KaTeX | O | O | O |
| JSON/YAML 편집 | O | O | O |
| 탭 수 | 3개 | 무제한 | 무제한 |
| 테마 | Classic만 | 4종 | 4종 |
| 집중 모드 | X | O | O |
| HTML/PDF 내보내기 | X | O | O |

### 8.2 라이선스 관리

- **키 형식:** `BPP-XXXXXXXX-XXXXXXXX-XXXXXXXX`
- **디바이스 ID:** SHA-256(hostname + seed), `bp-h-` 프리픽스
- **활성화:** 설정 > 라이선스에서 키 입력
- **비활성화:** Pro 사용자만 가능 (다른 PC로 이전 시)
- **오프라인 유예:** 마지막 검증 후 30일간 Pro 유지
- **트라이얼:** 최초 실행 시 자동 14일, localStorage 기반

### 8.3 다이얼로그

| 다이얼로그 | 시점 |
|-----------|------|
| 트라이얼 환영 | 최초 실행 (1회) |
| 트라이얼 만료 임박 | 잔여 3일 이하 (세션당 1회) |
| 트라이얼 만료 | 만료 후 (세션당 1회) |
| Pro 업그레이드 프롬프트 | Pro 기능 시도 시 |

## 9. 자동 업데이트

| 단계 | 상태 | 설명 |
|------|------|------|
| 1 | idle | 초기 상태 |
| 2 | checking | 서버 확인 중 |
| 3 | latest / available | 최신 버전 / 업데이트 존재 |
| 4 | downloading | 다운로드 중 (진행률 표시) |
| 5 | done | 설치 완료, 재시작 대기 |

- **엔드포인트:** Cloudflare Worker → R2에서 update.json 서빙
- **서명:** Tauri 자체 서명 (minisign)
- **업데이트 후 알림:** "What's New" 다이얼로그 (버전 비교)

## 10. 데스크톱 통합 (Tauri)

| 기능 | 설명 |
|------|------|
| 파일 연결 | .md/.txt/.json/.yaml 더블클릭으로 BluePad 열기 |
| CLI 열기 | `bluepad.exe file.md` 명령줄 실행 |
| 네이티브 다이얼로그 | 파일 열기/저장 OS 다이얼로그 |
| 외부 링크 | 시스템 브라우저에서 열기 |
| 자동 업데이트 | Tauri updater plugin |
| 앱 재시작 | 업데이트 설치 후 relaunch |

### Tauri 플러그인

| 플러그인 | 용도 |
|----------|------|
| tauri-plugin-dialog | 파일 열기/저장 다이얼로그 |
| tauri-plugin-fs | 파일 읽기/쓰기 |
| tauri-plugin-shell | 외부 URL 열기 |
| tauri-plugin-updater | 자동 업데이트 |
| tauri-plugin-process | 앱 재시작 |

### Rust 커맨드

| 커맨드 | 용도 |
|--------|------|
| `get_cli_file_path` | CLI 파일 경로 전달 |
| `get_hostname` | 디바이스 ID 생성용 호스트명 |
