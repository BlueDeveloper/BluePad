BluePad 일반 텍스트 테스트 파일
==================================

이 파일은 .txt 확장자로 열었을 때
SourceEditor(textarea)로 렌더링되는지 확인하기 위한 테스트입니다.

1. 마크다운 문법이 렌더링되지 않아야 함
   - **굵게** → 그대로 **굵게** 표시
   - *기울임* → 그대로 *기울임* 표시
   - # 제목 → 그대로 # 제목 표시

2. 탭 입력 테스트
	탭 문자가 여기 있습니다 (Tab key)
		두 번째 탭

3. 특수문자 테스트
   한글: 가나다라마바사
   일본어: あいうえお
   중국어: 你好世界
   이모지: 😀 🎉 ❤️ 🔥
   기호: @ # $ % ^ & * ( ) { } [ ] | \ / < > ~ `

4. 긴 줄 테스트
   이 줄은 매우 긴 텍스트를 포함하고 있어서 에디터에서 줄바꿈이 어떻게 처리되는지 확인할 수 있습니다. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
